# Dnyandeep Coaching Classes Website

A modern, professional, fully responsive, and highly optimized marketing and educational website for **Dnyandeep Coaching Classes**, Savarde, Hatkanangale, Kolhapur, Maharashtra.

Built purely using modern **HTML5**, **CSS3 (including custom styling panels)**, and **vanilla JavaScript**—no heavy frameworks, no external build tools, and completely static-ready for instant deployment on platforms like **GitHub Pages**.

---

## 🌟 Key Features

1. **Complete 11-Page Architecture**: Covers Home, About, Courses, Faculty, Results, Gallery, Testimonials, FAQ, Contact, Privacy Policy, and Terms & Conditions.
2. **Double-Filter Results Board**: Enables filtering of student toppers by both year (2025, 2024, 2023) and exam category using fluid, native vanilla JavaScript logic.
3. **Responsive Image Gallery with Custom Lightbox**: Grouped categories (Classroom Sessions, Infrastructure, Study Material, etc.) coupled with an in-house developed lightbox featuring swipe-ready navigation and escape keys.
4. **Interactive Hero Spotlight Carousel**: An auto-rotating student/parent spotlight carousel on the Testimonials page with manual back/next keys and active dot indicators.
5. **Interactive FAQ Accordion**: 15 custom-drafted, coaching-specific questions answering admissions, batch timings, installment structures, and assessment frequencies.
6. **Robust Inquiry Form**: Fully verified frontend form with built-in 10-digit mobile check, custom email pattern validation, and auto-constructed pre-filled WhatsApp click routing.
7. **Complete SEO & Offline Files**: Custom sitemap, robots.txt, descriptive metadata, Open Graph labels, and responsive layout classes for zero-scaling layouts.

## ⚡ Deployment Instructions

To deploy this project to **GitHub Pages**, follow these simple steps:

1. Create a new GitHub repository named `dnyandeep-classes`.
2. Initialize Git in this directory and commit all static files:
   ```bash
   git init
   git add .
   git commit -m "Initialize Dnyandeep Coaching Classes website v1.0.0"
   ```
3. Link and push the codebase to your remote repository:
   ```bash
   git remote add origin https://github.com/your-username/dnyandeep-classes.git
   git branch -M main
   git push -u origin main
   ```
4. Navigate to your repository settings on GitHub, select **Pages** from the sidebar, choose the `main` branch as your source, and click **Save**.
5. Your website will be live at `https://your-username.github.io/dnyandeep-classes/` in a few minutes!
